Flag to indicate if scan results should be viewable when a build has finished. When enabled, a new action will be provided to view scan results, labeled 'InsightAppSec Scan Results'.

Warning: All users with access to view the build job history will be able to see InsightAppSec scan results

Note: This is only applicable when 'Scan has been completed' or 'Vulnerability results query has returned no vulnerabilities' has been selected for build advance.
